RUNNING LAMPS

 To run LAMPS, open a terminal window, change to the directory where you want to work, and type lamps or lamps&. All LAMPS output files will be created in this directory. The input files (e.g. spectra or list-mode files for analysis) could be in other directories too. For ease of use, it is better to have the input files also in the same directory, but if working with many files, one could organise the files in sub-directories.

It is intended that the opening window, appears at the top of the screen. If it appears elsewhere (problems with some window managers), move the window to the top of the screen.

This window provides drop down menus for common tasks, a screen selector, refresh rate selector and status information about the running program.
