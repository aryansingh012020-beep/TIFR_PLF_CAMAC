Scalers

Clicking Scalers brings up a window displaying current values of all the defined scalers. The Total scaler value, the Rate and Diferential Rate (from the last acquired buffer) are displayed. When initially started the displays show "Pl. wait", then after the first buffer fills, the scaler values are displayed. The LAMPS main window already displays the values of the first 4 scalers. If there are more than 4 scalers defined, the values can be seen here.

We also draw your attention to the files periodic.log and scaler.log where scaler values are recorded.
