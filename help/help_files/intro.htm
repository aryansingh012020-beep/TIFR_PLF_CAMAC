LAMPS (Linux Advanced Multi Parameter System) is a CAMAC data acquisition and analysis package. CAMAC acquisition is from the new generation TCAMCON-95/CC-2000 transputer based controller supplied by Electronics Division, BARC.

It is in use at the Pelletron Laboratory, TIFR since August 2002. It is intended for use in large-scale experiments. LAMPS supports data acquisition from one/two CAMAC crates connected to IBM compatible PC via a dual port PCI card.

In offline use, it can handle experiments with upto 1000 parameters, 1000 1d-spectra and 500 2d-spectra. (If you need more parameters or spectra you can edit lamps.h and recompile).

LAMPS runs under any recent version of LINUX.
