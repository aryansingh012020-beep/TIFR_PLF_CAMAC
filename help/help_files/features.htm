FEATURES
  • Multi Window Graphical Application
  • Integrated package for CAMAC Data Acquisition with 1 or 2 crates and Offline Analysis of list mode and spectra files
  • Supports list mode data in various formats: zls, NSC Candle/Freedom, gls, Exogam
  • Auto calibration with common sources
  • Fully featured peak search and fit
  • Includes the analysis functions of the old programs ACT (Dos based) and ACOFF (Linux based) plus much more
  • Free form banana gates without restrictions on how to draw the polygon
  • Standard Pseudo Parameters
  • Allows to link user written routines in FORTRAN
  • Analysis can be done in batch mode leaving the system for long hours of unattended work
